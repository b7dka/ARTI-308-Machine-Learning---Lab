## 📌 Overview
This lab builds a **Machine Learning classification model** to predict whether a user will **click on an advertisement** based on demographic and online behavior features from an advertising dataset.

---

## 🛠️ Methodology
The workflow follows a standard ML pipeline:

- **Data Exploration:** checking dataset structure, summary statistics, and column names  
- **Exploratory Data Analysis (EDA):** visualizing relationships between features using histograms, jointplots, and pairplots  
- **Feature Selection:** defining input features (`Daily Time Spent on Site`, `Age`, `Area Income`, `Daily Internet Usage`) and target (`Clicked on Ad`)  
- **Train-Test Split:** separating data into training and testing sets  
- **Model:** Logistic Regression  
- **Training:** fitting the model on the training data  
- **Evaluation:** predictions and classification metrics:
  - Precision  
  - Recall  
  - F1-score  

---

## 📊 Results
- Built a complete **end-to-end classification model**  
- Analyzed user behavior and its impact on ad clicking  
- Evaluated model performance using classification metrics  
- Identified patterns between user activity and ad interaction  

---

## Conclusion
The model successfully predicts whether a user will click on an advertisement using a structured **Logistic Regression** workflow. The lab demonstrates how classification models can be used to analyze user behavior and support decision-making in digital marketing.
